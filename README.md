# BMI Calculator

A Flutter application for calculating Body Mass Index (BMI).

## Features

- **Gender Selection:** Choose between Male and Female.
- **Height Adjustment:** An interactive slider to select height in centimeters (cm).
- **Weight and Age Adjustment:** Easy-to-use buttons to increase or decrease weight (kg) and age.
- **Real-time BMI Calculation:** The BMI value updates automatically as you adjust any of the inputs.
- **Status Indication:** The output BMI number changes color dynamically based on the health category:
  - Blue: Underweight (BMI < 18.5)
  - Green: Normal weight (BMI 18.5 - 24.9)
  - Orange: Overweight (BMI 25 - 29.9)
  - Red: Obese (BMI >= 30)

## Getting Started

To run this project, make sure you have Flutter installed on your machine.

1. Clone or download the repository.
2. Open the project in your preferred IDE (like Android Studio or VS Code).
3. Run `flutter pub get` to install any necessary dependencies.
4. Run the app on your connected device or emulator using `flutter run`.
